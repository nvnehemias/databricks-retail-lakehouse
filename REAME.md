# Databricks Retail Lakehouse Analytics Project

## Overview
This project demonstrates an end-to-end retail analytics workflow using 
Databricks, PySpark, SQL, Delta Lake, Tableau, and GitHub.

The project simulates a real-world data engineering and analytics 
pipeline:
- Data ingestion
- Data cleaning
- Spark transformations
- Delta table creation
- SQL analytics
- Dashboard development

---

## Tech Stack
- Databricks
- PySpark
- SQL
- Delta Lake
- Tableau
- Git & GitHub
